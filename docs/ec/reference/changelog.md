# Changelog

Track all updates, changes, and additions to the guidelines documentation.

## 2025-12-16 - Initial Release

### Added
- 🎉 **Initial documentation site launch**
- Complete role-based documentation for Submitters, Reviewers, and Adjudicators
- Comprehensive workflow reference (6 steps)
- Detailed failure mode documentation (3 core categories)
- System prompt guidelines and templates
- Tool reference documentation (4 tools)
- FAQ & Glossary
- Internal documentation structure

### Site Features
- Role-based landing page navigation
- Search functionality (local search plugin)
- Deep-linkable headings
- Effective date and owners display on all pages
- Internal docs gating via INTERNAL_DOCS env var
- Mobile-responsive design

### Documentation Structure

**By Role:**
- Submitter: Overview, Checklist, Common Mistakes
- Reviewer: Overview, Rubric, Feedback Writing, Edge Cases
- Adjudicator: Overview, Tie-Breaking Principles, Edge Cases

**Reference:**
- Workflow: 6-step process from Intent to Final Review
- Failure Modes: Instruction Retention, Course Correction, Task Continuation
- System Prompts: Rules, Patterns, Templates
- Tools: web_search, news_search, open_url, open_search, Common Bugs
- FAQ & Glossary
- This Changelog

**Internal:**
- Overview, Calibration Notes, QA Process, Metrics, Release Notes

## How to Suggest Changes

Found an issue or have a suggestion?

1. **Documentation Issues:**
   - Submit via internal feedback form
   - Tag with "documentation"
   - Include page URL and specific suggestion

2. **Guideline Clarifications:**
   - Raise in calibration sessions
   - Document with examples
   - Discuss with team lead

3. **New Examples Needed:**
   - Note which area lacks examples
   - Provide draft examples if possible
   - Explain use case
